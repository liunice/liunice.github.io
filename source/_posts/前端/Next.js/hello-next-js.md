## 初识 Next.js

Next.js 是一个基于 React 的全栈开发框架，它提供了许多开箱即用的功能，让开发变得更加简单和高效。

### Next.js 的主要特性

- 服务端渲染 (SSR)
- 静态站点生成 (SSG)
- 文件系统路由
- API 路由
- 零配置
- 自动代码分割

### 快速开始

使用 create-next-app 创建项目：

```bash
npx create-next-app@latest my-next-app
```
